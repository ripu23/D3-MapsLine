Since, d3 is reading the csv and topojson file, the browser does not allow us to read a file locally. Hence, a server is needed to read it.Steps to open the file:1. Download the extension for chrome - https://chrome.google.com/webstore/detail/web-server-for-chrome/ofhbbkphhbklhfoeikjpcbhemlocgigb2. Choose the “RipudamanTeja” folder in the extension just downloaded by clicking on “CHOOSE FOLDER” in the extension3. Click on “Web Server URL(s)”.4. If index.html does not opens automatically, it can be manually be opened by clicking on the Web Server URLS(s) created and adding '/index.html' to the address.
